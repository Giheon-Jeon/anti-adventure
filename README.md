# anti-adventure 🛡️

기존 C언어 기반의 칙칙했던 CLI TRPG 로직을 **React 기반 웹 애플리케이션으로 전면 개편(Web Transition)**한 프로젝트임!  
단순한 텍스트 쪼가리에서 벗어나, 이제 브라우저에서 화려한 UI와 함께 주사위를 굴리며 진짜 모험을 즐길 수 있음.

---

## 🚀 주요 기능 (Main Features)

### 1. 야찌(Yahtzee) 스타일 다이내믹 주사위 전투 🎲
- **5 vs 3 주사위 굴리기**: 플레이어는 5개, 적은 3개의 주사위를 굴려 족보를 겨룸.
- **포커 족보 적용**: 풀 하우스, 스트레이트, 포카드, 파이브카드 등 주사위 조합에 따라 데미지가 최대 5배까지 폭증함!
- **선방어 후공격**: 적이 먼저 공격하는 쫄깃한 턴제 전투. 적의 맹공을 버텨내야 반격할 수 있음.

### 2. 10레벨 자동 전직 & 히든 직업 시스템 🎓
- **빅데이터 기반 전직 심사**: 10레벨에 도달하는 순간, 여태 올린 스탯, 장착 중인 무기 티어, 그리고 뒤에서 몰래 기록되던 온갖 플레이 로그(최저 체력, 돈 쓴 양, 몬스터 킬 수 등)를 종합 평가해 알아서 직업이 정해짐.
- **4단계 등급 (Common ~ Legendary)**:
  - `일반`: 전사, 도적, 마법사 (그리고 아무것도 안 한 자를 위한 '백수')
  - `희귀`: 검투사, 암살자, 성기사
  - `영웅`: 죽음의 문턱을 넘나든 '광전사', 만수르 뺨치는 '대도', 포션 중독 '정령술사' 등 히든 조건 필요!
  - `레전더리`: 극한의 운빨 '타짜', 모든 것을 마스터한 '용사' 등 극악의 조건.
- **정보 상인**: 도무지 영웅/레전더리 직업 조건을 모르겠다면? 마을의 정보 상인에게 500G를 찔러주면 은밀한 힌트를 던져줌.

### 3. 직업별 전용 스킬트리 ⚔️
- 전직 완료 시 해당 직업만의 **전용 스킬 3종 (1차, 2차, 궁극기)**이 해금됨.
- 스킬 포인트를 투자해 데미지 뻥튀기, 피흡, 방어력 무시, 주사위 주작 등 개사기 능력들을 마스터할 수 있음.

### 4. 다중 슬롯 & 유니온(계정 통합 보너스) 시스템 👥
- **최대 3개의 캐릭터 슬롯**: 한 계정에서 여러 캐릭터를 번갈아 가며 키울 수 있음.
- **유니온 보너스 효과**: 내가 키운 모든 캐릭터들의 '총합 레벨'이 현재 접속 중인 캐릭터의 전투력(모든 스탯 + 최대 체력)을 뻥튀기시켜줌!
- **직업 시너지**: 심지어 다른 슬롯에 특정 직업(예: 초보자)이 있으면 보너스 HP가 더 오르는 등 부캐 육성의 꿀맛을 느낄 수 있음.

### 5. 직관적인 웹 UI 시스템 (Web Transition) 💻
- **React 기반 뷰 컴포넌트**: 클릭 한 번으로 마을, 상점, 대장간, 던전을 오감.
- **돈 없으면 빨간불**: 골드가 부족하면 구매 버튼이 얄짤없이 빨간색(Danger)으로 변해서 지갑 사정을 직관적으로 알려줌.
- **실시간 상태 연동**: 전투, 훈련, 알바 등 모든 이벤트가 UI에 즉각 반영됨.

### 6. 경제 및 강화된 보상 체계 💰
- **마을 알바**: 푼돈 벌다 허리 삐끗할 수 있음. 운 좋으면 팁도 받음.
- **상점 & 여관**: 던전 뺑뺑이 후 지친 몸을 50G에 풀피로 채워주는 든든한 여관과 계단식으로 강해지는 장비 상점.
- **6단계 상위 던전**: 초심자 들판부터 뼈도 못 추리는 '드래곤의 둥지'까지 레벨업에 맞춰 더 가혹한 던전이 열림.

## 🛠 기술 스택 (Tech Stack)

![React](https://img.shields.io/badge/React-61DAFB?style=for-the-badge&logo=react&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)

- **Vite & React**: C언어 콘솔창 냄새를 싹 빼고 현대적인 SPA(Single Page Application)로 탈바꿈하기 위해 씀. 빠르고 쾌적함.
- **LocalStorage**: 별도 DB 없이 브라우저 저장소를 털어 세이브 데이터를 영구 보존함 (JSON 파싱).

```text
anti-adventure/
├── web/                    # 🌐 Web Client (React)
│   ├── src/
│   │   ├── components/     # UI 뷰 컴포넌트 (Village, Dungeon, Shop, Settings 등)

│   │   └── index.css       # 글로벌 스타일링 (Glassmorphism 적용)
│   └── package.json
├── trpg/                   # 🏰 Legacy C Engine (역사의 뒤안길로 사라진 구버전)
└── README.md
```

## ⚙️ 시작하기 (Getting Started)

### 웹 클라이언트 실행 (Web Version)
CLI 시절은 끝났음. 이제 아래 명령어로 세련된 웹 서버를 띄워보셈.
1. `web` 폴더로 진입함.
   ```bash
   cd web
   ```
2. 패키지 설치하고 바로 실행 때림.
   ```bash
   npm install
   npm run dev
   ```
3. 브라우저에서 `http://localhost:5173` 접속하면 모험 시작!

## 🎮 게임 흐름 (Game Flow)

1. **초보자의 삶**: 처음엔 약해 빠졌으니 알바와 훈련소로 푼돈 모아 장비 맞추셈.
2. **사냥 & 주사위 기도**: 던전에 들어가서 5개의 주사위가 제발 풀하우스 띄워주길 기도하며 몬스터를 줘팸.
3. **10레벨의 심판 (자동 전직)**: 10레벨을 찍는 순간 당신의 과거 행적에 따라 운명이 결정됨. 맘에 안 들면 '설정' 탭 가서 새 슬롯 파서 부캐 키우셈 (유니온 개꿀).
4. **전용 스킬 난사**: 전직 후 얻은 개사기 스킬로 상위 던전(드래곤의 둥지)을 찢으러 감.
5. **엔드 스펙**: 모든 슬롯 레벨 만렙 찍고 유니온 효과 최대로 땡긴 다음 '전설의 용사' 직업 띄우면 겜 끝!

## 🔧 최근 업데이트 이력 (v3.5 대격변 패치)

- **턴 순서 개편**: 예전엔 플레이어가 선빵 필승이었으나, 이제 **몬스터가 무조건 선제 공격**을 날리게 되어 방어력과 체력 관리의 중요성이 떡상함.
- **자동 전직 & 히든 클래스**: 단순 노가다에서 벗어나, 행동 로그 기반의 전직 시스템과 등급제 직업 추가.
- **다중 슬롯 & 유니온 떡상**: 1캐릭 게임에서 3캐릭 게임으로 확장. 본캐 몰빵보다 부캐 육성 시너지가 엄청남.
- **호환성 핫픽스**: 옛날 세이브 파일 불러오면 신규 던전/직업 로직 꼬이던 현상 완벽히 고침.
- **UI 직관성 개선**: 돈 없는데 버튼 누르고 멍때리지 않게 골드 부족하면 버튼 뻘겋게 칠해버림.
