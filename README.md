# anti-adventure 🛡️

Windows 터미널 환경에서 즐기는 **데이터 기반 C언어 TRPG** 프로젝트입니다.  
사용자의 활동과 선택에 따라 캐릭터가 성장하며, 16종의 다양한 전직 및 등급제 이벤트를 통해 다채로운 모험을 경험할 수 있습니다.

---

## 🚀 주요 기능 (Main Features)

### 1. 전직 및 성장 시스템
- **16종의 직업군**: 기초 스탯 분배에 따라 Single, Hybrid, Elite, 그리고 전설적인 **Legendary(Avatar)** 직업까지 전직이 가능합니다.
- **자동 스탯 성장**: 무작위 성장 알고리즘과 특정 활동(훈련, 지식 습득 등)을 연동한 스탯 강화 시스템을 갖추고 있습니다.

### 2. 전투 및 스킬 시스템
- **턴제 전투**: 적과의 긴장감 있는 턴제 공방을 지원합니다.
- **직업 전용 스킬**: 각 직업마다 고유한 공격 배율과 특수 효과를 가진 스킬 시스템이 구현되어 있습니다.
- **콤보 시스템**: 특정 조건에서 발동하는 강력한 연계 공격을 지원합니다.

### 3. 등급제 이벤트 시스템
- **무작위 인카운터**: 모험하기 메뉴를 통해 다양한 무작위 이벤트를 만날 수 있습니다.
- **등급 시스템**: Common, Uncommon, Rare, **Legendary** 등급으로 나뉘며 등급이 높을수록 파격적인 보상을 제공합니다.

### 4. 경제 및 마을 활동 (Hub)
- **상점 및 인벤토리**: 획득한 골드로 아이템을 구매하고 관리할 수 있습니다.
- **마을 활동**: 아르바이트를 통한 자금 확보, 도박 시스템, 아이템 제작 등 전투 외적인 풍성한 콘텐츠를 포함합니다.

## 🛠 기술 스택 및 선정 이유 (Tech Stack)

| 기술 | 구분 | 선정 이유 |
|:---:|:---:|---|
| **C Language** | Core | 포인터, 구조체 등 메모리 관리 기초와 절차적 프로그래밍의 핵심 로직 구현 |
| **Windows CMD** | Environment | ANSI Escape Code를 활용하여 별도의 그래픽 라이브러리 없이 텍스트 UI(TUI) 구현 |
| **Batch Script** | Build | GCC 컴파일러를 활용한 멀티파일 프로젝트의 빌드 프로세스를 자동화하여 개발 생산성 향상 |
| **UTF-8 (65001)** | Encoding | Windows 환경에서의 한글 깨짐 문제를 해결하고 다국어 호환성 확보 |

## 🏗 아키텍처 및 폴더 구조 (Architecture)

본 프로젝트는 **모듈화**를 최우선으로 설계되어 기능별로 파일이 분리되어 있습니다.

```text
anti-adventure/
├── trpg/
│   ├── src/                # 핵심 로직 소스 코드 (.c)
│   │   ├── main.c          # 프로그램 진입점 및 메인 루프
│   │   ├── combat.c        # 전투 엔진 logic
│   │   ├── character.c     # 캐릭터 스탯 및 전직 관리
│   │   └── skill.c         # 직업별 스킬 및 콤보 데이터
│   ├── include/            # 인터페이스 정의 및 헤더 파일 (.h)
│   │   ├── combat.h        # 전투 관련 구조체 및 함수 선언
│   │   └── character.h     # 플레이어/몬스터 데이터 구조 정의
│   └── build/              # 컴파일된 실행 파일 저장 경로
├── build.bat               # 빌드 자동화 스크립트
└── README.md               # 프로젝트 문서
```

## ⚙️ 시작하기 (Getting Started)

### 사전 준비
- Windows OS 환경
- GCC 컴파일러 (MinGW 등) 설치 및 환경 변수 설정

### 빌드 및 실행
1. 저장소를 클론하거나 다운로드합니다.
2. 프로젝트 루트 폴더에서 `trpg` 폴더로 이동합니다.
3. 터미널(CMD)에서 아래 명령어를 실행합니다.
   ```bash
   build.bat
   ```
4. 생성된 `build/game.exe`를 실행하여 모험을 시작하세요!

## 🎮 게임 흐름 (Game Flow)

1. **캐릭터 생성**: 초기 능력치가 무작위로 부여됩니다.
2. **마을 활동**: 훈련으로 스탯을 올리거나 상점에서 장비를 정비합니다.
3. **모험 & 사냥**: 던전에서 적을 물리치고 경험치를 획득하거나, 무작위 이벤트를 겪습니다.
4. **전직**: 특정 조건(레벨, 스탯)을 만족하면 상위 직업으로 전직하여 강력한 스킬을 얻습니다.
5. **목표**: 전설적인 아바타 직업에 도달하고 최강의 용사가 되는 것이 목표입니다.
